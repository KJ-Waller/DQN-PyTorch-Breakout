# DQN-PyTorch-Breakout
## Overview
An attempt at recreating DeepMind's implementation of Deep Q Learning on Atari Breakout using PyTorch.

## Dependencies
* PyTorch
* [OpenAI Gym](https://github.com/openai/gym)
* Pillow
* Numpy
